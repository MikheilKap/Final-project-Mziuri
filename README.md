# გამოწერის გაუქმების ახსნა (AI API)

FastAPI-ზე დაფუძნებული უკანა ნაწილი, რომელიც Groq-ის ენობრივ მოდელს იყენებს გამოწერის გაუქმების ნაბიჯების მარტივად ახსნისთვის.

## რას აკეთებს

1. კლიენტი აგზავნის სერვისის სახელს და გაუქმების ნაბიჯების სიას.
2. სერვერი აგებს პრომპტს (`prompt_service`) — მხოლოდ მოწოდებული ნაბიჯების ახსნა, ახალი ინფორმაციის გამოგონების გარეშე.
3. Groq API აბრუნებს მომხმარებლისთვის გასაგებ ახსნას.
4. პასუხი ფორმატირდება JSON-ად (`formatter`) და ბრუნდება კლიენტზე.

## მოთხოვნები

- Python 3.10 ან უფრო ახალი
- [Groq](https://console.groq.com/) ანგარიში და API გასაღები

## API

### `POST /ai/explain-cancel`

**მოთხოვნის სხეული (JSON):**

```json
{
  "service": "Netflix",
  "steps": [
    "გადადით ანგარიშის პარამეტრებში",
    "დააჭირეთ გამოწერის გაუქმებას",
    "დაადასტურეთ გაუქმება"
  ]
}
```

**წარმატებული პასუხი (მაგალითი):**

```json
{
  "service": "Netflix",
  "explanation": "... AI-ის მიერ დაწერილი ახსნა ...",
  "steps": [
    "გადადით ანგარიშის პარამეტრებში",
    "დააჭირეთ გამოწერის გაუქმებას",
    "დაადასტურეთ გაუქმება"
  ]
}
```

**შეცდომები:**

| კოდი | მიზეზი |
|------|--------|
| 422 | არასწორი JSON ან ველი აკლია |
| 500 | `API_KEY` არ არის დაყენებული |
| 502 | Groq API-თან პრობლემა |
